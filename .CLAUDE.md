## Project Overview
An enterprise-grade, multi-agent AI pipeline for automated video and ad compliance auditing.
- **Tech Stack :** Python 3.12, FastAPI, LangGraph, Azure OpenAI, Azure AI Search, Azure Video indexer, PostgreSQL, Redis.

## Build, Environment, & Execution Commands
Always use 'uv run' to execute scripts or servers to ensure the local '.venv' context is respected. Do not use global 'pip' or plain 'python'.

### Enviorment Management
- Create virtual enviorment: 'uv venv .venv'
- Sync/install dependencies: 

### Running the Application
- Run FastAPI development server: 'uv run uvicorn backend.app.main:app --reload'
- Run Celery worker queue: 'uv run celery -A backend.app.core.celery_app worker --loglevel=info'

### Testing & Linting
- Run all unit and integration tests: 'uv run pytest'
- Run a specific test file: 'uv run pytest tests/test_agents.py'
- Lint code with Ruff: 'uv run ruff check .'
- Auto-fix linting issues: 'uv run ruff check . --fix'
